WhatIm.at

Individualized Data For Diabetics

WhatIm.at is a website dedicated to the presentation and application of research for advancing intelligent type 1 diabetes management using machine learning. Focused on improving insulin therapy recommendations for physicians and patients using time-series analysis.

The research will culminate in two parts. The first part will be a supervised machine learning algorithm capable of making recommendations for insulin therapy with insulin pumps and the artificial pancreas. The second part will be a web based application. Patients and physicians can use the application to upload data sets from various sources to take advantage of the algorithm developed in part one.

This project is supported by Thomas Hastings, a graduate student, at the University of Colorado at Colorado Springs. If this area of research interests you and you would like to collaborate, please contact me.
